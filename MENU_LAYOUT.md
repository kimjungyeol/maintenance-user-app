# 메뉴 레이아웃 구조 설명

## 개요
이 문서는 정비관리 앱의 헤더 및 네비게이션 메뉴 레이아웃 구조를 설명합니다.

## 전체 구조

### 헤더 컨테이너
- **위치**: `position: sticky, top: 0, zIndex: 100`
- **목적**: 스크롤 시 상단에 고정되는 헤더 영역
- **구성 요소**:
  - 헤더(Header)
  - 모바일 네비게이션 메뉴(Mobile Navigation)

## 데스크톱 환경 (1024px 이상)

### 헤더 레이아웃
```
┌─────────────────────────────────────────────────────────┐
│  정비관리          [메뉴1] [메뉴2] [메뉴3] ...          │
│  (좌측)                   (중앙)                        │
└─────────────────────────────────────────────────────────┘
```

### 상세 구조
- **헤더**: `display: flex, justifyContent: center, position: relative`
- **타이틀 "정비관리"**:
  - `position: absolute, left: 16px`
  - 폰트 크기: 20px, 굵기: bold
  - 색상: `var(--primary-color)`
- **네비게이션 메뉴**:
  - 중앙 정렬 (flex center)
  - 가로 방향 배치 (`display: flex`)
  - 메뉴 항목 간격: 8px
  - 클래스: `.desktop-only`

## 모바일/태블릿 환경 (1024px 미만)

### 헤더 레이아웃 (메뉴 닫힘)
```
┌─────────────────────────────────────────────────────────┐
│  정비관리                                        ☰      │
│  (좌측)                                        (우측)   │
└─────────────────────────────────────────────────────────┘
```

### 헤더 레이아웃 (메뉴 열림)
```
┌─────────────────────────────────────────────────────────┐
│  정비관리                                        ☰      │
│  (좌측)                                        (우측)   │
├─────────────────────────────────────────────────────────┤
│  - 홈                                                   │
│  - 매출 관리                                            │
│  - 지출 관리                                            │
│  - 미수금 관리                                          │
│  - 급여 관리                                            │
│  - 월별 정산                                            │
│  - 기초 설정                                            │
└─────────────────────────────────────────────────────────┘
```

### 상세 구조
- **헤더**: `display: flex, justifyContent: center, position: relative`
- **타이틀 "정비관리"**:
  - `position: absolute, left: 16px`
  - 모바일에서도 동일한 스타일 적용
- **햄버거 버튼**:
  - `position: absolute, right: 16px`
  - 클래스: `.mobile-only`
  - 크기: 44x44px (터치 친화적)
- **드롭다운 메뉴**:
  - **중요**: `position: absolute` (콘텐츠를 밀어내지 않음)
  - `top: 100%` (헤더 바로 아래)
  - `left: 0, right: 0` (전체 너비)
  - 애니메이션: `maxHeight` 전환 (0 ↔ 400px, 0.3s)
  - 그림자: `boxShadow: 0 4px 6px rgba(0, 0, 0, 0.1)`
  - 클래스: `.mobile-only`

## 반응형 클래스

### .mobile-only
- 기본: `display: block`
- 1024px 이상: `display: none`

### .desktop-only
- 기본: `display: none`
- 1024px 이상: `display: block`

## 메뉴 항목 구성

### 네비게이션 메뉴 항목
1. 홈 (`/`)
2. 매출 관리 (`/sales`)
3. 지출 관리 (`/expenses`)
4. 미수금 관리 (`/receivables`)
5. 급여 관리 (`/payroll`)
6. 월별 정산 (`/monthly-report`)
7. 기초 설정 (`/settings`)

### 메뉴 항목 스타일
- **활성 상태**:
  - 배경색: `var(--primary-color)`
  - 텍스트 색상: `#fff`
  - 폰트 굵기: 500
- **비활성 상태**:
  - 배경색: `transparent`
  - 텍스트 색상: `var(--text-color)`
  - 폰트 굵기: 400

## 주요 특징

### 1. 콘텐츠 밀림 방지
- 모바일/태블릿 메뉴가 `position: absolute`로 설정되어 있어 메뉴가 열릴 때 페이지 콘텐츠가 밀려나지 않음
- 메뉴는 콘텐츠 위에 오버레이됨

### 2. Sticky 헤더
- 헤더 컨테이너가 스크롤 시 상단에 고정됨
- `z-index: 100`으로 다른 요소 위에 표시

### 3. 상태 관리
- `isMenuOpen` state로 모바일/태블릿 메뉴 토글 상태 관리
- 메뉴 항목 클릭 시 자동으로 메뉴 닫힘

### 4. 반응형 브레이크포인트
- **1024px 미만**: 모바일/태블릿 환경 (햄버거 메뉴)
- **1024px 이상**: 데스크톱 환경 (가로 메뉴)

## 파일 위치
- 메인 컴포넌트: `src/App.tsx`
- 스타일: `src/index.css`

## 수정 시 주의사항
1. 모바일/태블릿 메뉴의 `position: absolute` 속성을 유지해야 콘텐츠 밀림이 발생하지 않음
2. 반응형 클래스(`.mobile-only`, `.desktop-only`)는 **1024px** 기준으로 동작
3. 타이틀과 햄버거 버튼은 `position: absolute`로 좌우에 고정
4. 데스크톱 메뉴는 `justifyContent: center`로 중앙 정렬
